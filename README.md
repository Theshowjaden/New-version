## Deployment Guides

### Vercel
1. Sign up or log in to Vercel.
2. Import your GitHub repository.
3. Set environment variables if necessary.
4. Click on "Deploy".

### Netlify
1. Sign up or log in to Netlify.
2. Click on "New site from Git".
3. Connect your GitHub repository.
4. Set build command as `npm run build` and publish directory as `dist`.
5. Click "Deploy site".

## Troubleshooting for npm run build Error 127
If you encounter `Error: Command failed with exit code 127`, please ensure to:
1. Check if all dependencies are correctly installed.
2. Verify that required scripts are present in the `package.json`.
3. Consult the logs for more details on the error.
